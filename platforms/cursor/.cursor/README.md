# Archived Cursor Asset

This directory stores the archived Cursor-facing asset layout maintained by this repository.

It contains the distributable/backend-oriented Cursor asset set that used to live at the repository root. Repo-maintenance-only Cursor entrypoints now live in the root `.cursor/` directory and are intentionally not duplicated here.

## Contents

- `backend-commands/`
- `devops-commands/`
- `go-backend-commands/`
- `go-backend-rules/`
- `python-rules/`
- `project-specific-archived/`
- `user-commands/`
- `skills/`
- `agents/`

## Notes

- Treat this directory as archived external content under maintenance.
- If a change is meant only for maintaining this repository, put it in the root `.cursor/` directory instead.
