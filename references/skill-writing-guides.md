# Skill Writing Guidance Links

This file records high-signal references for creating or revising `SKILL.md` assets in this repository.

Keep this list short. Prefer official guidance and ecosystem-level references over opinionated blog posts.

## Primary References

### OpenAI Codex

- Link: <https://developers.openai.com/codex/skills>

## Repository Usage Rule

When this repository creates or revises a `SKILL.md` asset, review these links first and keep only the instructions and bundled resources that are:

- aligned with the official skill structure and behavior
- specific enough to guide repeatable agent work
- scoped to the skill instead of duplicating repo-level governance
