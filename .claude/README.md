# Claude Repository Entry

This `.claude/` directory exists only to help maintain this repository.

Archived external Claude assets live under `platforms/claude/`.

## Scope

- repo-local workflow skills for maintaining this repository with Claude Code
- no distributable external platform assets
- English-first governance and maintenance documentation

## Skills

- `brainstorming` — explore intent and design before implementation
- `git-start-task` — start a clean task branch from updated main
- `git-commit` — create a focused, well-formed commit
- `pr-handoff` — prepare a review-ready PR handoff
- `pr-operator` — push branches and manage GitHub PR operations via bundled `scripts/prctl`
- `repo-doc-simplifier` — compress bloated repo-maintenance Markdown without changing meaning
- `repo-self-iteration` — entry skill for goal-driven repo-local maintenance tasks
- `prompt-sync` — preview and run manifest-driven platform asset sync and backflow
- `migrate-codex-to-claude` — migrate repo-local Codex maintenance assets to Claude
